# Coursework 1 – [Computer Vision and Pattern Recognition]

## Overview
This repository contains the code, report, and supporting materials for **Coursework 1** of the module **Computer Vision and Pattern Recognition**.

## Cloning and Setup
1. Create a folder called Coursework
2. Place the unzipped dataset folder PR_CW_Dataset_2026 inside it
3. Git clone this repository into the Coursework folder
